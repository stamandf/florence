The Chat App

The Chat App allows a user to type in a display name, create a chat or join multiple chat rooms. When a member joins for the first time, a Bot sends a welcome message. Members of each chat room can send messages to each other, share their current location as well as images.
Technology used: Expressjs, Nodejs and Socket.io.
Third party libraries: moment.js, mustache.js, jQuery, deparam.js.
Unit testing: mocha.js and expect.js.
